# ToolioAI - Free Online Multi-Tool Website

A fully client-side multi-tool website with 15+ useful tools, built with pure HTML, CSS, and JavaScript.

## Features

- **15+ Tools**: Text summarizer, grammar checker, password generator, QR code generator, and more
- **100% Client-side**: All processing happens in the browser
- **No Backend Required**: Can be deployed on GitHub Pages
- **Dark/Light Theme**: Automatic theme switching
- **Fully Responsive**: Works on all devices
- **SEO Optimized**: Proper meta tags and schema markup
- **AdSense Ready**: Pre-built ad placements
- **Fast & Lightweight**: No external dependencies

## Tools Included

1. Text Summarizer
2. Grammar & Spell Checker
3. Text Rewriter/Paraphraser
4. Password Generator
5. QR Code Generator
6. Image Compressor
7. PDF Merger
8. Unit Converter
9. Word & Character Counter
10. Base64 Encoder/Decoder
11. Color Picker & Palette Generator
12. Image to Base64 Converter
13. Random Name Generator
14. HTML/CSS/JS Minifier
15. Todo List (localStorage)
